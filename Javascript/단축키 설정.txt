[설치한 Extensions]

1. Code Runner

Ctrl + Alt + N 

위 단축키를 통해, 콘솔에서 출력 log를 볼 수 있다. (결과 출력)
-> console.log(" ")와 같은 식으로 출력해야 한다.
(위 로그 방식은 브라우저에서는 보이지 않는다.)
(alert() 함수를 쓰면 에러가 난다.)


2. Live Server

Alt + Live

위 단축키를 통해, 브라우저의 local에서 실시간 구동이 가능하다.
-> console.log()같은 것은 브라우저 특성상 볼 수 없다.
-> Ctrl + S로 저장할 때마다, 자동 새로고침 되어서 매우 편리하다.
